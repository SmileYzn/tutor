<p>Plugin commands</p>

| Command                             | Parameters                                      | Description                                     |
| :---------------------------------- | :---------------------------------------------- | :---------------------------------------------- |
| cs_tutor_show                       | <name or #userid> <0~8> <hold_time> <text_msg>        | Send the message in notifiction tutor           |
| cs_tutor_hide                       | <name or #userid>                               | Hide current player tutor message               |
